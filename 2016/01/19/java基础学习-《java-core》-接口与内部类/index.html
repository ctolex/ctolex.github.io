java基础学习-《java-core》-接口与内部类

java core基础，接口与内部类

接口

  1. 接口中的所有方法都自动地属于public，因此接口中声明方法时，不必提供关键字public。不过，在实现接口时，必须把方法声明为public。

  2. 接口中可以包含常量，并且接口中的域将被自动设为public static final。

  3. 一个类只能拥有一个超类，但是却可以实现多个接口，使用,将实现的各个接口分隔开。
