2017年设定的几个目标

2016年的目标完成得不尽如人意，新的一年又到了，开启新的目标，加油！

工作

  • 提高效率，留有时间提升自己
  • 看书《maven实战》、《spring实战》、《深入理解java虚拟机》、《git版本控制管理》，有些看过部分，查漏补缺
  • 看源码 tmf2、logback、ateye

生活

  • 找个姑娘谈个奔婚姻的恋爱
  • 慎独、自律，提高个人修养
  • 照顾好自己

读书

  • 每月读两本书，一年24本书，读书需要有读书笔记
  • 读经典书籍，好书需要反复读
  • 读投资方面的书，需要慢慢形成自己的投资理念
  • 读书需要注意跨学科、跨界整合

健身

  • 跑步1000公里，平均每周20公里
  • 每天锻炼，不跑步就keep

思考

  • 从日常生活中找到机会
  • 保持耐心
  • 寻求“内心的宁静”
